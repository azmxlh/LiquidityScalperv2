WHEN TO CHOOSE MT4
================================================================================

✓ Your broker only offers MT4
✓ You're doing FTMO or similar prop challenge (MT4 required)
✓ You're familiar with MT4 and don't want to learn MT5
✓ You want maximum broker choice worldwide
✓ You're using older VPS or computer (MT4 less resource-intensive)
✓ You need compatibility with other MT4 indicators/tools
✓ You prefer simpler, more straightforward platform

MT4 ADVANTAGES:
- Wider broker support
- Simpler to learn
- More third-party tools available
- Established platform (proven over 15+ years)
- Lower system requirements

MT4 DISADVANTAGES:
- Slower execution
- Less accurate backtesting
- Older technology
- Limited to 9 timeframes
- Basic reporting tools

================================================================================
TECHNICAL DIFFERENCES (For Programmers)
================================================================================

CODE STRUCTURE:

MT4 (MQL4):
- Simpler syntax
- Direct array access (Close[1], High[2])
- Order-based model
- Easier to modify for beginners

MT5 (MQL5):
- C++-like syntax
- Copy functions (CopyHigh, CopyClose)
- Position-based model
- More powerful, but complex

TRADE EXECUTION:

MT4:
OrderSend() → returns ticket
OrderClose() → closes order
OrderModify() → modifies order

MT5:
CTrade.Buy/Sell() → opens position
CTrade.PositionClose() → closes position
CTrade.PositionModify() → modifies position

Both work perfectly for this EA's needs.

MIGRATION GUIDE (MT4 → MT5)
================================================================================

IF YOU'RE SWITCHING FROM MT4 TO MT5:

STEP 1: Understand the Differences
- Read Installation_Guide_MT5.txt
- Familiarize yourself with MT5 interface
- Note: Strategy is identical, just execution differs

STEP 2: Install MT5 Version
- Download MT5 from your broker
- Install EA files following MT5 guide
- Use SAME parameters as MT4 version

STEP 3: Parallel Testing (Recommended)
- Run MT4 version on demo
- Run MT5 version on demo
- Compare for 1-2 weeks
- Verify similar results

STEP 4: Transition
- Once confident, switch to MT5 live
- Keep same risk settings initially
- Monitor for 1 week
- Adjust if needed

EXPECTED TRANSITION TIME: 1-2 weeks

COMMON MIGRATION ISSUES:

1. "Things look different"
   → Yes, MT5 interface is different. Spend time exploring.

2. "My indicators don't work"
   → MT4 indicators won't work in MT5. This EA doesn't need them.

3. "I see hedging mode option"
   → Choose "Netting" mode for this EA (Tools → Options → Trade)

4. "Partial closes behave differently"
   → MT5 has native support, it's actually better!
