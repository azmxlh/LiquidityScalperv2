================================================================================
STEP 2: INSTALL EA FILES
================================================================================

METHOD A - Automatic Installation:

1. Open MetaTrader 4
2. Click File → Open Data Folder
3. Navigate to: MQL4 → Experts
4. Copy "LiquidityScalper_MT4.mq4" into the Experts folder
5. Navigate to: MQL4 → Include
6. Copy "LS_Functions.mqh" and "LS_Config.mqh" into the Include folder
7. Restart MetaTrader 4

METHOD B - Manual Installation:

Windows:
C:\Users\[YourUsername]\AppData\Roaming\MetaQuotes\Terminal\[BrokerID]\MQL4\

Mac (with Wine):
~/Library/Application Support/Wine/...

1. Place .mq4 file in: MQL4\Experts\
2. Place .mqh files in: MQL4\Include\
3. Restart MT4

================================================================================
STEP 3: COMPILE THE EA
================================================================================

1. In MT4, press F4 or click Tools → MetaQuotes Language Editor
2. In the Navigator panel (left side), expand "Experts"
3. Double-click "LiquidityScalper_MT4.mq4"
4. Click the "Compile" button (or press F7)
5. Check the "Errors" tab at the bottom
6. You should see: "0 error(s), 0 warning(s)"
7. If errors appear, verify all .mqh files are in the Include folder

✓ Successful compilation message:
  "LiquidityScalper_MT4.ex4 compiled successfully"

================================================================================
STEP 4: CONFIGURE MT4 SETTINGS
================================================================================

ENABLE ALGO TRADING:

1. Click Tools → Options
2. Go to "Expert Advisors" tab
3. CHECK the following:
   ☑ Allow automated trading
   ☑ Allow DLL imports
   ☑ Allow imports from external experts
   ☑ Allow WebRequest for listed URL (add: https://www.forexfactory.com)
4. Click OK

ENABLE AUTO TRADING BUTTON:

1. Look at the top toolbar
2. Click the "AutoTrading" button (should turn green)
3. If it's grayed out, your broker may not allow EAs

================================================================================

1. Open a new chart: File → New Chart → XAUUSD

4. Drag "LiquidityScalper_MT4" onto the XAUUSD M5 chart

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: EA doesn't open any trades

SOLUTIONS:
- Check if you're within trading hours (08:00-12:00 or 13:30-17:00 UTC)
- Verify daily levels are not zero (check Experts log)
- Ensure max trades per day not reached
- Check if daily loss limit was hit
- Verify news filter isn't blocking trades
- Make sure account has sufficient free margin

---

PROBLEM: Stop loss is too wide / EA rejects trades

SOLUTIONS:
- Market volatility may be too high
- Adjust "Max stop loss" parameters
- Check if liquidity sweep is far from order block
- Review recent price action for extreme volatility

---

PROBLEM: Too many losing trades in a row

SOLUTIONS:
- This is normal! System expects 55-70% win rate
- Do NOT increase risk to "recover losses"
- Take a break after 3 consecutive losses
- Review trade entries in backtest
- Consider reducing trade frequency (increase displacement minimum)

---

PROBLEM: Dashboard not showing / levels not drawing

SOLUTIONS:
- Right-click chart → Properties → Common tab
- Check "Show object descriptions"
- Increase max bars in chart (Tools → Options → Charts)
- Manually run: ObjectsDeleteAll() then restart EA

---

PROBLEM: Error 130 (Invalid stops)

SOLUTIONS:
- Your broker may have minimum stop distance
- Check broker specifications (Market Watch → Right-click Symbol → Specification)
- Increase "Min displacement points" parameter
- Contact broker to verify XAUUSD trading conditions

---

PROBLEM: Error 134 (Not enough money)

SOLUTIONS:
- Reduce risk per trade to 0.5% or lower
- Increase account balance
- Close other open trades
- Check required margin for XAUUSD
